# EarningsEdge

A Chrome extension for momentum swing traders. Research episodic pivots, earnings breakouts, and catalyst-driven moves — all in one dashboard.

Built for traders who study Qullamaggie, Stockbee, and Minervini-style setups.

## Features

- **Earnings History** — 8 quarters + 4 future estimates with EPS, revenue, surprises, and YoY growth
- **Earnings Calendar** — Weekly view with BMO/AMC timing, market cap filters, sortable columns
- **Analyst Ratings** — Consensus rating, price targets, upgrades/downgrades table
- **Insider Transactions** — Buys, sells, transaction values, position changes
- **News** — Aggregated from multiple sources with filters
- **TradingView Chart** — Embedded chart with responsive sizing

## Installation

1. Download the latest release from [Releases](https://github.com/earningsedgedev/earningsedge/releases)
2. Unzip the file
3. Open Chrome → go to `chrome://extensions`
4. Enable **Developer mode** (top-right toggle)
5. Click **Load unpacked** → select the `EarningsEdge-Extension` folder
6. Click the EarningsEdge icon in the toolbar to open

## Usage

Type any US stock ticker (e.g. NVDA, AAPL, TSLA) and press Enter. Use keyboard shortcuts for fast navigation:

- `Space` or `⌘K` — Focus search
- `1-6` — Switch tabs
- `Ctrl+R` — Force refresh
- `C` — Open earnings calendar

## Privacy

EarningsEdge collects anonymous usage analytics (search count, feature usage) to improve the extension. No personal data, no ticker history, no identifying information is collected. See the Docs page inside the extension for full details.
