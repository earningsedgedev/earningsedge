## Quick Start

```
1. Download the latest .zip from Releases
2. Unzip ‚Üí you get an EarningsEdge-Extension folder
3. Go to chrome://extensions ‚Üí enable Developer mode
4. Click "Load unpacked" ‚Üí select the folder
5. Pin it to your toolbar. Done.
```

Works on any Chromium browser (Chrome, Edge, Brave, Arc).

## What's Inside

### üìä Stock Dashboard
Search any US ticker and get **7 tabs** of data instantly:

**Earnings** ¬∑ **News** ¬∑ **Analysts** ¬∑ **Insiders** ¬∑ **Funds** ¬∑ **Short Interest** ¬∑ **Analysis**

The Earnings tab shows a TradingView chart alongside 8 historical quarters + 4 forward estimates with EPS, Revenue, %Change, and %Surprise ‚Äî all color-coded.

A **key stats bar** runs across the top: Price, Mkt Cap, P/E, Float, Short Float, ADR%, Rel Vol, 52W High (green when within Minervini's 10% zone), Inst Own, and more.

### üìà ETF Dashboard
Search an ETF (SMH, XLK, GDX‚Ä¶) and it auto-detects:

**Overview** ¬∑ **Holdings** ¬∑ **News** ¬∑ **Analysis**

Performance across 9 timeframes, top 15 holdings with price changes, and click-through to any holding's stock dashboard.

### üîç Four Scanners

| Scanner | What it does |
|---------|-------------|
| **Movers** | Top 200 gainers/losers across pre-market, market hours, and after-hours |

### üìÖ Earnings Calendar
Weekly view with BMO/AMC grouping, market cap filter, ADR%, and adaptive price columns that switch between pre-market, intraday, and after-hours data automatically.
