Get Started

Install EarningsEdge

Install manually in under a minute. Source code available on GitHub.

1

Download

Download the latest release from our GitHub Releases page and unzip the file.

2

Load in Chrome

Go to chrome://extensions, enable Developer mode (top right), then click Load unpacked and select the unzipped folder.

3

Start Trading

Click the EarningsEdge icon in your toolbar, type any ticker, and get instant access to earnings, analysts, insiders, and news.
